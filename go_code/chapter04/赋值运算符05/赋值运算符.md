#  赋值运算符

```bash
赋值运算符就是将某个运算后的值 赋给指定的变量
赋值运算符分类
```

#### 赋值运算符的分类

| 运算符 | 描述                                           | 实例                                 |
| ------ | ---------------------------------------------- | ------------------------------------ |
| =      | 简单的赋值运算符，将一个表达式的赋值给一个左值 | C = A  + B 将  A +B 表达结果赋值给 C |
| +=     | 相加后再赋值                                   | C += A 等于 C = C + A                |
| -=     | 相减后再赋值                                   | C -=  A 等于 C= C - A                |
| *=     | 相乘后再赋值                                   | C *= A 等于 C = C * A                |
| /=     | 相除后再赋值                                   | C /= A 等于 C =C / A                 |
| %=     | 求余后再赋值                                   | C  %= A 等于 C = C % A               |
| <<=    | 左移后赋值                                     | C <<= 2 等于 C = C << 2              |
| >>=    | 右移后赋值                                     | C >>= 2 等于 C = C >> 2              |
| &=     | 安位与后赋值                                   | C &= 2 等于 C= C & 2                 |
| ^=     | 按位异或后赋值                                 | C ^= 2 等于 C = C ^ 2                |
| \|=    | 按位或后赋值                                   | C \|= 2 等于 C = C \| 2              |
