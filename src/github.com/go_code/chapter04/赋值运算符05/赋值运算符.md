# 赋值运算符
    赋值运算符就是将某个运算后的值 赋给指定的变量
    赋值运算符分类
赋值运算符的分类

    | 运算符        | 描述  |  势力  |
    | --------   | -----   | ---- |
    | =          |   简单的赋值运算符，将一个表达式的赋值给一个左值     |   C=A+B 将 A+B表达结果赋值给c   |
    | +=        | 相加后再赋值      |   C+=A 等于 C=C+A   |
    | -=        | 相减后再赋值      |   C-=A等于C=C-A   |
    | *=        | 相乘后再赋值      |   C*=A等于C=C*A   |
    | /=        | 相除后再赋值      |   C/=A等于C=C/A   |
    | %=        | 求余后再赋值      |   C%=A等于C=C%A   |
